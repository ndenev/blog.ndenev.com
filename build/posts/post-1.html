Toeplitz RSS Hash

Symmetrical RSS using Toeplitz hash function.

While playing with some Rust code to parse Suricata EVE Json log events I was thinking about calculating and keeping per flow list of events (looks like Suricata flow id fields are not very unique, however this is changing). I found this article explaining the RSS Toeplitz hash function and a modification used to allow for symmetric hashing: Symmetric RSS

Here is also some DragonFlyBSD and FreeBSD code implementing Toeplitz:

  • https://github.com/freebsd/freebsd/blob/master/sys/net/toeplitz.c
